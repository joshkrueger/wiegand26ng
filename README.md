Wiegand26ng
===========

Arduino library for reading from Wiegand26 compatible devices.

Dependencies
============

Arduino >= 1.0

### For Non-Mega Arduinos
###PCATTACH
Interrupt library provided in [Open Access Control][oac] (Only if you want to use multiple readers)


Attribution
===========
